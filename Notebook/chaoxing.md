# 超星学习方法

> 这里记录的一些超星学习的方法，适用于浏览器端

## 视频的保存方式

有些课程使用视频进行讲解，但是超星中的视频很难进行调整进度，所以我选择将视频保存起来

1. 用浏览器登录网站后找到视频
2. 点击F12将浏览器工具打开
3. 找到网络（network）
4. 此时播放视频，会在network中出现一些元素
5. 选中含有mp4的元素右键复制链接，这个链接就是可以直接下载的视频文件

## 视频的跳转方式

大部分课程的一项评分标准就有在超星平台的视频学习检查点

1. 用浏览器登录网站后找到视频

2. 点击F12将浏览器工具打开

3. 使用快捷键Ctrl+F打开浏览器工具的搜索框，搜索fast

4. 找到彩色的元素，将fastforword删除干净，此时就可以任意拖动进度条

   > 注：在跳转到下一个视频时进度条就不能拖动了
   >
   > 目前本人还没有较好的解决方式
   >
   > 使用的是清除cookie
