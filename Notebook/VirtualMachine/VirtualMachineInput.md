# 问题描述

在安装完成Ubuntu后，把系统装成了中文版，导致文件目录出现中文，但是没有对应的输入法，急需中文输入方式

## 环境描述

在VMware16pro中安装的Ubuntu20.04.1 LST（几乎全新的系统）

本人选择使用国民输入法-->搜狗

## 安装步骤

1. 用浏览器打开[搜狗输入法的官网](https://pinyin.sogou.com/linux/?r=pinyin)
2. 点击立即下载（doge）
3. 用tar解压并安装
4. 打开设置，找到语言支持，添加中文
5. 键盘输入法系统改为fcitx（这里指free Chinese input toy for X）

## 使用

1. 在安装完成后，重启系统
2. 切换输入法的方法：用Ctrl+space
